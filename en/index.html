StarWright's Eye (Cosmoteer Tool)

A tool for Cosmoteer currently under development, which allows for detailed analysis of ship designs directly from the .ship.png files. The key lies in the method Cosmoteer uses to store ship data and how "StarWright's Eye" makes this data accessible:

  • Steganography in PNGs: Cosmoteer embeds ship data not obviously, but directly into the pixels of the .ship.png image files using steganography.
  • Data Extraction: The tool reads these embedded binary data from the color channels of the image pixels.
  • Processing Chain: The extracted data, which is in a proprietary binary format, is first decompressed using GZip. Subsequently, this binary data is converted into a readable format (similar to JSON) to enable the identification and counting of individual ship modules.

Further details on the analysis functions and features of "StarWright's Eye" will be announced at a later date.
